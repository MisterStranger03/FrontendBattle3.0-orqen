# orqen-platform - ✅ Round 1 Qualified

> Built for **Frontend Battle 3.0** · Qualified for the Finale (Round 2) on 28th June 2026.

A landing page built for **Frontend Battle 3.0 - Competition** hosted by IIT Bhubaneswar on [Unstop](https://unstop.com/hackathons/frontend-battle-30-vibecoding-competition-indian-institute-of-technology-bhubaneswar-1696294).

## About the Competition

Frontend Battle 3.0 is a two-round frontend development hackathon:

- **Round 1 (Qualifying)** - Build a functional landing page using pre-defined assets in 4 hours, implementing specific feature requirements and SEO optimization.
- **Round 2 (Finale)** - 12-hour final round on Unstop (details revealed to qualifiers only).

## Tech Stack

- TanStack Start + React 19
- Vite 8
- Tailwind CSS v4
- Three.js + React Three Fiber (3D hero scene)
- TypeScript

## Getting Started

```bash
npm install
npm run dev
```

## Build
